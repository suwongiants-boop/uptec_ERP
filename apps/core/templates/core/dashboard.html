먼저 기존 문서 보관소에 카테고리별 원본 문서를 등록하고, 문서 생성 코너에서 해당 카테고리의 생성 문서와 기존 문서를 자동 참조해 새 문서를 만들 수 있습니다.

문서 생성 방식

1. 문서 카테고리 선택
2. 생성 문서와 기존 문서 보관소 자동 참조
3. 재사용 섹션과 신규 생성 섹션 확인

관리 포인트

승인된 블록이 많을수록 재사용률이 높아집니다.
영업기획과 연결된 문서 흐름을 같이 관리하세요.
